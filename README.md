# Stream Trailer Simulation

A photorealistic and interactive 3D stream trailer simulation built with Next.js and React Three Fiber.

## Features

- Interactive terrain modification (dig, fill, sand)
- Water flow simulation
- Vegetation placement (trees, grass, bridges)
- Real-time erosion effects

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the simulation.
